# ShitBit Logo Prompt

Use this prompt with an image generation tool (Midjourney, DALL-E, etc.)

## Primary Logo

```
Minimalist app icon logo for "ShitBit" — a smart bathroom scale product.
Clean, modern, premium tech aesthetic. Dark background (#0A0A0F).

The logo is a stylized toilet seat viewed from above, forming a circular/oval
shape, with a small digital readout or graph line integrated into the design.
Subtle weight/scale motif. Color palette: amber/gold (#C68B2C) as the accent
on dark charcoal.

Think Fitbit or Oura Ring branding but with a toilet seat silhouette hiding
in the geometry. It should look like a legitimate fitness/health tech logo
at first glance — the joke only lands when you look closer.

No text in the icon. No poop emoji. No cartoon style. Premium, minimal,
geometric. Suitable for iOS and Android app icons.
```

## Wordmark Variant

```
Clean sans-serif wordmark reading "ShitBit" in a modern tech font.
The "i" in "Bit" is dotted with a small circle that subtly resembles
a toilet seat or scale dial. Dark background, amber/gold text.
Premium tech branding. Could be mistaken for a real Silicon Valley
startup logo. That's the point.
```

## Brand Colors

- Primary dark: `#0A0A0F`
- Accent amber: `#C68B2C`
